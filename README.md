# ScloudJS　v0.0b
nodejsを使用して、Scratchのクラウド変数を監視、変更ができるライブラリです。

## 必要なもの
node.js 18.0+
## 使い方
1. ファイルをダウンロード
```
git clone https://github.com/xxXFreezerXxx/ScloudJS.git
```
or Download zip

2. パッケージをインストールする
```
npm init -y
npm install ws
```
3. `index.js`を書き換える
projectid,username,passwordを自分のものに書き換える<br>
`tasks()`の関数の中に自分好みのコードを書く
